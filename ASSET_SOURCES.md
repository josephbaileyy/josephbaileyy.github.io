# Room social asset sources

The room uses these third-party brand and game assets only to identify Joseph's
profiles in a personal fan portfolio. The marks and artwork remain the property
of their respective owners; their inclusion does not imply affiliation or
endorsement.

| Local asset         | Source                                                                                                                                                |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| `instagram-glyph.*` | [Meta Instagram Brand Resources](https://about.meta.com/brand/resources/instagram/instagram-brand/)                                                   |
| `letterboxd-dots.*` | [Letterboxd brand assets](https://letterboxd.com/about/brand/)                                                                                        |
| `fortnite-omega.*`  | Epic Games' Omega cosmetic render, retrieved through the [Fortnite-API cosmetic endpoint](https://fortnite-api.com/v2/cosmetics/br/search?name=Omega) |
| `league-katarina.*` | Riot Games [Katarina splash art](https://ddragon.leagueoflegends.com/cdn/img/champion/splash/Katarina_0.jpg)                                          |
| `clash-princess.*`  | Supercell's [Princess Evolution update](https://supercell.com/en/games/clashroyale/blog/release-notes/june-update-2026/)                              |

Modern WebP files are the primary runtime assets. The source-format PNG/JPEG
files are retained as compatibility fallbacks.
